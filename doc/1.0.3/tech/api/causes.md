## Все пользователи

```http
GET causes
```

### Возвращаемые значения

Возвращает массив объектов типа [`Cause`](resources/cause)

### Пример ответа

```json
[  
    {
        "id": 106,
        "title": "Причинение вреда другим лицам случайное (по неосторожности)",
        "category": "Правонарушения",
        "points": -15,
        "access": [
            "teacher"
        ]
    }
]
```
